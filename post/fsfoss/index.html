intro

FSF

Free software is software that gives you the user the freedom to share, study and modify it. We call this free software because the user is free.

Free Software Foundation(自由软件基金会)立志于赋予用户分享、学习、修改软件的自由。你可以在FSF上面找到超过1.5w个自由软件，例如GIMP，IceCat等。

Free software

根据GNU关于自由软件的描述，自由软件应当拥有以下几个基本条件

  • The freedom to run the program as you wish, for any purpose (freedom 0).
  • The freedom to study how the program works, and change it so it does your computing as you wish (freedom 1). Access to the source code is a precondition for this.
  • The freedom to redistribute copies so you can help others (freedom 2).
  • The freedom to distribute copies of your modified versions to others (freedom 3). By doing this you can give the whole community a chance to benefit from your changes. Access to the source code is a precondition for this.

GNU

GNU操作系统是由自由软件基金会支持下开发的操作系统。

OSS

Open Source Software(开源软件)的缩写.详细的定义可以查看Open Source definition 开源软件一般具有以下特征：

  1. It is available in source code form (without charge or at cost)
  2. Open Source may be modified and redistributed without additional permission
  3. Finally, other criteria may apply to its use and redistribution. 在第二点的时候用的是may而不是可以，这就是后面众多开源协议的开端，想修改传播商用取决于开源软件所使用的开源协议。在对用户友好性上来说，FS更优于OS，有时候，FS和OS并称为FOSS - Free and Open Source。
