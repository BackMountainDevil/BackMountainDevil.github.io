# 联想小新风扇控制
- date: 2022-10-18
- lastmod: 2022-10-18

在 windows 上有官方驱动可以通过 Fn+Q 调节电脑模式（性能、平衡、省电），这个设置也可以在 BIOS 里面设置，但是在 linux 下这种快捷键由于没有驱动就实现不了这样的操作。
