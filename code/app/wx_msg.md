# 留痕

[使用教程](https://memotrace.cn/doc/posts/deploy/)
> 仅支持Windows微信3.9.x版本

1. 登录电脑微信，手机微信发起迁移功能，迁移完成后重新登录微信
2. 打开 MemoTrace，工具栏目，点击获取信息，解析数据
3. 聊天栏目，选择对应的账号，右上角导出

实测简便易用，含图片、语音、视频。点个赞。
